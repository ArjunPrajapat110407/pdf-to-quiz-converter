# सेटअप गाइड

इस दस्तावेज़ में आपके सिस्टम पर प्रोजेक्ट को स्थापित और सेटअप करने की प्रक्रिया समझाई जाएगी।

## आवश्यकताएँ
- Python 3.x
- pip (Python पैकेज मैनेजर)

## स्थापित करने के लिए कदम
1. सबसे पहले, इस प्रोजेक्ट को क्लोन करें:
   ```bash
   git clone https://github.com/ArjunPrajapat110407/pdf-to-quiz-converter.git
   ```

2. प्रोजेक्ट डायरेक्टरी में जाएं:
   ```bash
   cd pdf-to-quiz-converter
   ```

3. आवश्यक पैकेज स्थापित करें:
   ```bash
   pip install -r requirements.txt
   ```

4. प्रोजेक्ट को चलाने के लिए, निम्नलिखित कमांड का उपयोग करें:
   ```bash
   python main.py
   ```

## निष्कर्ष
इन कदमों का पालन करके, आप आसानी से प्रोजेक्ट को स्थापित कर सकेंगे।

यदि आपको किसी भी प्रकार की समस्या आती है, तो कृपया [issues](https://github.com/ArjunPrajapat110407/pdf-to-quiz-converter/issues) पृष्ठ पर जाएं।
